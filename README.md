# Bible_Alias 🎮

Біблійний Аліас — гра в слова на біблійну тематику для Christian Youth Groups, церковних заходів та домашнього кола. Пояснюйте слова своїй команді та набирайте бали!

## Особливості

- 📖 **8 категорій слів**: книги, люди, міста, події, свята, притчі, об'єкти, місця
- 👥 **Підтримка команд** (2–8 гравців)
- ⏱ **Налаштовуваний таймер** та кількість раундів
- 🎯 **3 рівні складності**: Легко, Про, Потрібно напрягтися
- 🌙 **Темна/світла тема**
- 🌍 **Українська та англійська мови**
- 📱 **Telegram Mini App** — грайте прямо в Telegram
- 📊 **Статистика** ігор та серій

## Як грати

1. Розділіться на команди (мінімум 2)
2. Один гравець пояснює слово, не називаючи його
3. Команда відгадує — проведіть картку вгору ✅
4. Не знаєте слово — проведіть вниз ⏭
5. За кожне вгадане слово — 1 бал
6. Команда з найбільшою кількістю балів перемагає!

## Швидкий старт (локально)

Просто відкрийте `index.html` у браузері — все працює одразу, без сервера.

## Деплой на Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/zeroprotocolx86/Bible_Alias)

Або вручну:
1. Форкніть/клонуйте цей репозиторій
2. Зайдіть на [Netlify](https://app.netlify.com) → Import from GitHub
3. Виберіть цей репозиторій
4. `netlify.toml` вже налаштований — натисніть Deploy

## Telegram Mini App

1. Створіть бота через [@BotFather](https://t.me/BotFather)
2. Встановіть домен вашого Netlify-сайту: `/setdomain`
3. Налаштуйте Menu Button з URL вашого застосунку

## Оновлення бази слів

1. Відредагуйте файли в `data/`
2. Закомітьте та запуште
3. Netlify передеплоїть сайт автоматично
